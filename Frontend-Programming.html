The frontend refers to the client side of a website. Front-End programming, therefore, is concerned interactive functionality to the client side of a webpage. This includes buttons, animations, and some local storage. Most companies use JavaScript and TypeScript for frontend programming

TypeScript

Typescript is a superset of Javascript. This means that any js code will also run in ts. Typescript provides additional features that javascript doesn't have. Including OOP support, Static Typing, Type Annotations, and other features.
